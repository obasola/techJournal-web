This application provides functionality and features to aide in tracking Agile development. Below is a list of features that give a hint to how the application can add value to team or individual development efforts.

  • Record agile stories and sprint ticket assignments
  • Keep track of Source Code changes and the reason for the changes
  • SQL script DDL and DML
  • Batch Jobs and the input files used for test execution
  • Pull it all together to create report summaries and details to support historical views of work performed
  • Search by date range to create report of weekly contributions (Brag sheet)
